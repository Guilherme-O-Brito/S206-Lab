# Testes Cypress - E-commerce Playground

## Instalação das dependências
Instale o Cypress e o Mochawesome:
```bash
npm install cypress mochawesome --save-dev
```

## Iniciar o Cypress
Abra o painel interativo do Cypress:
```bash
npx cypress open
```

## Gerar relatório de testes (Mochawesome)
Após rodar os testes:
```bash
npm run test
```

O relatório HTML será gerado em:
```
mochawesome-report/mochawesome.html
```
